GitHub set up guide

Create a free GitHub account.

  1. Open a browser and head to www.github.com.
  2. Next click on the sign up button on the top right hand corner.
  3. At this page, you can create a username and password. Once you have done this click on "create account"
  4. Follow the instruction to finish creating your account. Make sure to select the free account instead of the paid account. Note: the free account is public, if you want a private repository you have to get the paid version.

Download Desktop Git

  1. Go to www.desktop.github.com, and select the download for your operating system. Alternatively you can click the links provided to download for you system: Mac OS X 10.9+ windows 7+
